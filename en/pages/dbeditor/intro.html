# Intro



# 1、What is DB Editor

DB Editor is a special editing tool for modifying the FIFA16 Mobile game database. You can use it to modify a lot of data in the game, such as player name, ability value, transfer, weight, height, team and so on. You only need to use DB Editor to complete the operations of modifying, saving, importing and exporting these data.

# 2、Why use DB Editor

  • DB Editor is a modification tool customized for the FIFA16 Mobile game database, so it has perfect adaptability and reliability.
  • The DB Editor provides a visual editing panel, which is very convenient to modify and browse data.
